# Requirements

- Laravel 9
- PHP  >= 8
- Npm = 9.5.0

#Fitur

- CRUD petugas
- CRUD kelas
- CRUD siswa
- CRUD Spp
- Authentikasi petugas

#Langkah - langkah instal project

- <b> Clone project </b>
- composer install / composer update
- npm install
- php artisan key:generate
- <b> kemudian set up database </b>
- php artisan migrate
- php artisan db:seed
- php artisan storage:link

#Jalankan project

- php artisan serve
- npm run dev
- login or register
